# WitnessChain

Digital notary and witness verification.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/WitnessChain.sol`
- Stacks: `contracts/stacks/contracts/witnesschain.clar`

## Setup
```bash
pnpm install
pnpm dev
```
